## Nginx
這是一套開源的 web server 服務，其中包含了反向代理、負載平衡、郵件代理（mail proxy）、HTTP 快取等功能，知名的使用者包括 Dropbox, Netflix 和 Zynga。

## Apache HTTP Server
又稱作 Apache，是另一個非常熱門的開源免費 web server。

## Apache Tomcat
主要提供 Java application server 使用

## Glassfish
主要提供 Java application server 使用
