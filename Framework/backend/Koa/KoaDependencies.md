# 套件安裝指令
## 安裝指定版本
```shell
npm i <套件名稱>@<指定版本>
```
## 套件版本更新
```shell
npm update
```
## 在開發、發佈時都需要依賴的套件
```shell
npm i <套件名稱> --save
```
## 在開發時才會依賴的套件，發佈後不需要使用
```shell
npm i <套件名稱> --save-dev
```
