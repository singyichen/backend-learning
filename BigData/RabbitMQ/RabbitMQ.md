## 比較表
| Message Broker | RabbitMQ | Kafka |
|:--:|:--:|:--:| 
| 架構 | 傳統訊息佇列 | 分散式串流平台 |
| 用途 | 可靠的訊息傳遞、交易式資料處理 | 高吞吐量資料擷取、即時資料串流 |
| 訊息傳遞模式 | 點對點、發布訂閱、路由 | 發布訂閱 |
| 儲存 | 記憶體、磁碟 | 磁碟 |
| 訊息順序 | 嚴格保證 | 通常保證 |
| 延遲 | 低 | 較高 |
| 可擴展性 | 有限 | 高度可擴展 |
| 容錯性 | 高 | 非常高 |
| 社群活躍度 | 高 | 非常高 |
